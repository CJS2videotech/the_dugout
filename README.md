# The Dugout - UI Tracker

A custom, stylized Cubs & Brewers live baseball tracker featuring native MLB API telemetry, dynamic team-based viewer themes, and a responsive Emoji Reaction Engine.

## GitHub Pages Deployment
1. Go to github.com and create a new repository called 	he-dugout.
2. Drag and drop **all files** inside this folder directly into your new repository.
3. Commit the changes.
4. Go to the repository **Settings** tab.
5. Click **Pages** on the left sidebar.
6. Under Build and deployment, make sure Source is set to **Deploy from a branch**.
7. Set the branch to **main**, keep the folder as **/ (root)**, and click **Save**.
8. In a few minutes, GitHub will give you a live URL to share your app!
